# Список сутностей та пояснення зв'язків 

# Layer 

PR: numerical order який він за порядком вниз 

amount of souls inside: скільки душ зараз знаходиться всередині


# Soul

PR: ID унікальний ключ для кожної душі

Full name:  повне ім'я і тільки ім'я

sin(s): повний список всіх гріхів людини

date  of arrival: дата прибуття в ад 

# Demon

PR:ID унікальний ключ для кожного демона

Full name: повне ім'я 

Age: вік

Status: статус демона в ієрархії


# punisment

PK:ID унікальний ключ для кожного покарання

required sin: гріх, який необхідний для виконання цього покарання

performing demon: статус демона, необхідний для виконання цього покарання (низший, вищий, верховний)

# пояснення зв'язків

layer - soul: одна душа має бути лище в одному слої, але багато душ може бути в одному слої одночасно

layer - demon: один демон має бути лище в одному слої, але багато демонів можуть бути в одному слої одночасно

demon - punisment: багато демонів можуть виконувати багато або нуль покарань

layer - punisment: на одному слої може бути багато покарань, але деякі з них можуть бути лише на одну слої
